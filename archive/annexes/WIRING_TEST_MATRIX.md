## Purpose

This document enumerates every "must-be-wired" integration path in the NZT-48 trading system. A wiring path is any data flow that crosses a module boundary where a break would produce silent failure, stale outputs, or inconsistent state across delivery channels. Every path listed here must be tested before any deployment and after any code change that touches the components involved.
