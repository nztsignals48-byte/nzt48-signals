## 11.1 Bayesian Stranger Penalty

```
kappa(n, DSR) = kappa_min + (kappa_max - kappa_min) x f_DSR(DSR) x f_n(n)

f_DSR(DSR) = 1 - exp(-lambda x max(0, DSR - DSR_min))
f_n(n) = n / (n + n0)

Parameters: kappa_min=0.25, kappa_max=1.0, lambda=0.5, DSR_min=1.5, n0=50
Graduation: t-stat >= 3.0 (HLZ 2016) AND P(mu_edge > 0 | data) > 0.98
```
