WHAT HAPPENS AUTOMATICALLY
================================================================================

Session Runs Until One of These (Whichever Comes First):

1. 50 Trades Closed
   → Validation gates evaluated
   → If all pass: ✅ Ready for LIVE
   → If any fail: ❌ Halt, analyze, restart

2. 14 Days Elapsed
   → Time limit reached (data gathering)
   → Gates evaluated at current status
   → Either LIVE-ready or needs fixes

3. Daily Heat ≤-4% (Heat Cap)
   → Circuit breaker triggered
   → Session halts to prevent spiral
   → Manual review required before restart

4. Any Gate Fails (After 5 Trades)
   → Entry Quality <60%, or
   → Rung Hit Rate <60%, or
   → Win Rate <60%, or
   → Profit Factor <1.5x, or
   → Max Cascades ≥3
   → → Session halts, analysis required

5. Manual Stop
   → Ctrl+C in terminal, or
   → kill <pid>, or
   → touch data/KILL_SWITCH
